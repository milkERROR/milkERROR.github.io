重装SQL Server

删除干净SQL Server

1.关闭所有服务

搜索-服务，停止所有SQL Server的服务

2.卸载SQL Server数据库

控制面板-卸载程序，先找到Microsoft SQL Server 2012(64-bit) 删除实例，有多少个实例就要删除多少次。再卸载所有SQL Server相关程序。

3.删除相关文件

默认情况下：

4.删除SQL Server注册表

win+r输入regedit

5.重启电脑

安装SQL Server2025

参考视频：https://www.bilibili.com/video/BV1fQUrBBEys/

1.下载SQL2025-SSEI-StdDev.exe

下载Developer版本

安装的是custom版本

2.进入SQL Server 2025安装中心

选择第一个 - 全新SQL Server独立安装或向现有安装添加功能。

基本都是默认选项，功能选择前三个安装；身份严重模勾选混合模式，设置一下管理员密码，添加当前用户为sql server管理员。

ps.安装最后报错：等待数据库引擎回复句柄失败

解决办法⬇

https://www.cnblogs.com/ChengHB/p/16013323.html https://blog.csdn.net/qq_43421954/article/details/127976167

3.回到安装中心，选择安装SQL Server管理工具

（第三个）跳转到 https://learn.microsoft.com/zh-cn/ssms/install/install

选择表格中的”下载SSMS 22”直接下载即可。

选择ai辅助和代码工具，手动勾选Help Viewer。
